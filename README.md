# Swiss Budget Calculator 

This application is a financial tool designed for Swiss budget calculation and savings planning. It provides users with a precise overview of net income and living expenses based on specific parameters for various Swiss cantons.

## Features

- Tax and contribution calculations based on specific cantons (Zurich, Geneva, Bern, Vaud, Valais).
- Monthly and Yearly calculation modes.
- Toggle for 13th-month salary inclusion.
- Detailed expense tracking (Rent, Health Insurance, Transport, Groceries, etc.).
- Financial stability analysis (AI Health Score).
- Long-term wealth projections with an 8% expected market return.
- Executive report export functionality via PDF.

## Technologies

- HTML5: Application structure.
- CSS3: Full responsive design for mobile and desktop devices.
- JavaScript: Financial logic and DOM manipulation.
- Chart.js: Visual representation of budget allocation.
- jsPDF: PDF report generation.

## Usage

1. Clone the repository to your local machine.
2. Open the index.html file in any modern web browser.
3. Select a canton and enter a gross salary amount to begin the analysis.
